# Personal Portfolio Website

## 🚀 Overview

This is a personal portfolio website showcasing my professional work, skills, and experiences. The site is built with cutting-edge web technologies and provides an interactive and responsive platform to highlight my projects and capabilities.


## 🔗 Live Demo

Check out the live portfolio: [visit](https://harshanabatagalla.github.io/portfolio/)


## 🌟 Features

- **Responsive Design**: Fully optimized for all device sizes and screen resolutions
- **Modern Tech Stack**: 
  - React for dynamic and interactive user interfaces
  - Tailwind CSS for sleek, custom styling
  - Three.js for immersive 3D graphics
  - Framer Motion for smooth animations
- **Contact Integration**: EmailJS for seamless communication
- **SEO Optimized**: Structured for improved search engine visibility

## 🚦 Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/harshanabatagalla/portfolio.git
```

2. Install dependencies
```bash
cd portfolio
npm install
```

3. Run the development server
```bash
npm run dev
```
